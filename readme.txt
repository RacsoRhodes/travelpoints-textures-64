	About

		64x64 textures for the Minetest mod "Travelpoints".

	Install

		Copy or move the image files from this directory to the Minetest texture
		pack you are currently using.

		For example, if you are using "HDX 64", you would move these files to
		the textures/HDX_64px directory in your Minetest installation directory.

	Travelpoints on GitHub

		https://github.com/RacsoRhodes/travelpoints
